Das Pin Game ist ein Puzzle, das aus einem Brett mit kreuzförmigen Löchern besteht, die anfangs mit Stiften gefüllt sind, bis auf ein leeres Loch in der Mitte. Das Ziel ist es, die Stifte mit gültigen Bewegungen zu entfernen, bis nur noch ein Stift übrig ist. Ein Zug ist gültig, wenn eine Stecknadel (Drehpunkt) über eine benachbarte Stecknadel (Ziel) in Richtung eines leeren Lochs (Ziel) springen kann, wodurch die Zielstecknadel entfernt wird.

Sie erhalten eine Spielbrettkonfiguration als Liste von 7 Zeichenfolgen, die jeweils eine Zeile auf dem Spielbrett darstellen. Jede Zeichenkette enthält genau 7 Zeichen, wobei '-' (Bindestrich) in den ersten beiden und letzten beiden Zeichen der Zeilen 1, 2, 6 und 7, 'o' (Kleinbuchstabe) für einen Stift oder '.' (Punkt) für ein Loch steht.

Ihr Programm sollte die Anzahl der möglichen gültigen Züge ermitteln.
